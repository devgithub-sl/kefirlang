# Contributing to KefirLang

Welcome! KefirLang is an open-source project, and **the language is yours to modify**. 

Whether you want to add new syntax, optimize the interpreter, or fix bugs, your contributions are welcome. We believe in keeping the core simple while robustifying features for new learners.

## ðŸ›  How to Contribute

1.  **Fork the Repository:** Click the "Fork" button at the top right of this page to create your own copy of the project.
2.  **Clone your Fork:**
    ```bash
    git clone [https://github.com/YOUR_USERNAME/kefirlang.git](https://github.com/YOUR_USERNAME/kefirlang.git)
    cd kefirlang
    ```
3.  **Create a Branch:**
    ```bash
    git checkout -b my-new-feature
    ```
4.  **Make Changes:** Hack away! Add that new keyword you always wanted.
5.  **Test:** Use the built-in REPL or upload `.kf` files to ensure your changes work.
6.  **Commit & Push:**
    ```bash
    git commit -m "Added a cool new feature"
    git push origin my-new-feature
    ```
7.  **Open a Pull Request:** Go to the original repository and open a PR describing your changes.
